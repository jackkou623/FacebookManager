Facebook Manager is a powerful application with GUI made to manage large-scale of facebook accounts. It provides rich configurations to the following functions:
- login/signup
- sharing
- posting
- actions scheduling
- group list updating

Leveraged technologies:
- MySQL (sql file is not included here, you have to make your own, suggest GCP, you can inform me to provide you a db template)
- Qt
- Python
- Selenium
- Chrome & chrome driver (not included in this REPO, please download the updatest - https://chromedriver.chromium.org/downloads)
